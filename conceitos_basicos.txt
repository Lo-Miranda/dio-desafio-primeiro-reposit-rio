# Git

Conceitos Básicos:

. Git é encriptado (SHA1)

---

**COMANDOS CLI**

| Geral | cls (clear screen) |
| --- | --- |
| Mudar de pastas | cd (change directory)
. Exemplo: cd Windows
. Dois pontos (..) depois de cd retrocede na árvore de diretórios. |
| Listar pastas | dir (lista diretórios do sistema) |
| Criar pastas/arquivos | mkdir (make directory) |
| Deletar pastas/arquivos | . del (para arquivos)
. rmdir (remove directory) |

---

**GIT COMANDOS**

| Para ver a versão do GIT | git—version
Se der erro ele não está instalado corretamente. |
| --- | --- |
| Iniciar | git init
Certifique-se de estar na pasta certa do projeto que vc quer lidar |
| Adicionar um arquivo da pasta para o repostiório via gitbash | git add . ou git -A |
| Ver SHA1 |  | Ver SHA1 do arquivo
